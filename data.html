Published on Jan 29, 2017

A quick introduction to Spacemacs, a community driven configuration for the worlds longest running editor come operating system, Emacs.

This short video shows how to go from zero to Clojure coding without any manual configuration (well except for changing the font). Hopefully this shows just how easy Emacs is to use when you start with Spacemacs.

The menu system used in the video is opened using the Space key, `SPC`, when in Vim command mode or `M-m` in Emac or any other modes.

Please note: if the Clojure layer mode doesn't automatically download the packages and run Clojure mode, you have found a bug (spacemacs version 200.7). To resolve this, either restart Emacs with `SPC q r`, or use `SPC f e R` to download the Clojure layer and `SPC SPC clojure-mode` to change to Clojure mode.

Learn more about Spacemacs at the project website http://spacemacs.org and take a look at my workshop on Clojure development with Spacemacs (work in progress)

https://practicalli.github.io/spacemacs
